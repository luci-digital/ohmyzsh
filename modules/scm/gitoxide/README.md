# scm/gitoxide

Gitoxide is consumed two ways, so this dir is a reference point, not a vendor copy:
1. As the `gix` crate dependency of `../luci-vcs` (pure-Rust git engine).
2. Embedded as `gix`/`ein` binaries inside the Gogs image (`../gogs/Dockerfile.gogs`).

Ref: https://github.com/GitoxideLabs/gitoxide
